# Project Overview

This project presents an interactive sales dashboard built using Python to analyze and visualize sales data. It helps understand sales trends, product performance, regional distribution, and the relationship between price, quantity, and total sales through static and interactive charts.

# Objectives

Analyze sales data using Python
Identify trends and patterns in sales
Compare performance across products and regions
Visualize insights using Seaborn and Plotly
Build a dashboard suitable for business reporting

# Technologies Used

Python
Pandas – Data manipulation and analysis
Matplotlib & Seaborn – Static visualizations
Plotly – Interactive charts
